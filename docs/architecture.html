Factory Inventory Management System

Tech Stack

Frontend

Vue 3 Composition API Vue Router Vite Axios

SPA with reactive composables, i18n (EN/JP), and custom SVG charts. Port 3000.

Backend

Python FastAPI Pydantic Uvicorn

REST API with auto-generated Swagger docs, CORS enabled, in-memory filtering. Port 8001.

JSON Files In-Memory

7 JSON files loaded at startup into memory. No database — data resets on server restart.

Data Flow (Request Lifecycle)

1
User interacts with FilterBar (period, warehouse, category, status) or navigates to a page.
2
Composable watcher fires. useFilters refs change, triggering watch() in the active view.
3
API call dispatched. api.js builds URLSearchParams and sends GET request via Axios.
4
FastAPI receives request. Query params parsed, apply_filters() and filter_by_month() run against in-memory data.
5
Filtered JSON returned. Pydantic validates response shape. Sent as JSON over HTTP.
6
Vue ref updated. Component ref() receives data. computed() properties re-derive (search, sort, charts).
7
Template re-renders. Reactive DOM updates via Vue's virtual DOM diffing. i18n and currency formatting applied.

State Management (Composables)

useFilters

Singleton reactive state for 4 global filters. Provides getCurrentFilters(), resetFilters(), and hasActiveFilters computed. Shared across all views via import.

useI18n

Locale management (EN/JP) persisted to localStorage. Translation helper t(), product/warehouse name translators, auto currency switching (USD/JPY at 150:1).

useAuth

Mock user data with locale-aware name. Provides currentUser, getInitials(), logout(), and mock task list for TasksModal.
